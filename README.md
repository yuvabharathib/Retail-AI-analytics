# Retail AI Analytics Platform

## Project Overview

An end-to-end Retail AI Analytics platform built using Python, PostgreSQL, AWS S3, Machine Learning, and Power BI.

The project automates data cleaning, stores processed data in PostgreSQL, predicts product sales using Machine Learning, and visualizes insights through an interactive Power BI dashboard.
